Calculus

Learn limits, continuity, differentiation, integration, and their applications through clear explanations and solved examples.

What is Calculus?

Calculus is the branch of mathematics that studies change and accumulation. It is mainly divided into Differential Calculus and Integral Calculus.

Main Topics

  • Limits

Why Learn Calculus?

Calculus is essential in mathematics, engineering, physics, economics, computer science, data science, and artificial intelligence.

Limits

The foundation of calculus.

A limit describes the value that a function approaches as the input approaches a particular point.

Example:

lim (x → 2) (x² + 3) = 7

Substitute x = 2: 2² + 3 = 4 + 3 = 7

Continuity

Understanding continuous functions.

A function is continuous at a point if:

  1. The function is defined at that point.
  2. The limit exists at that point.
  3. The limit equals the function value.

A continuous function has no breaks, holes, or jumps.

Differentiation

Finding the rate of change.

Definition

Differentiation measures how rapidly a function changes with respect to its variable.

Example

If f(x)=x² then f′(x)=2x

At x = 3,

f′(3)=6

So the slope of the curve at x = 3 is 6.

Integration

The reverse process of differentiation.

Definition

Integration is used to find areas under curves and recover a function from its derivative.

Example

∫2x dx = x² + C

where C is the constant of integration.

Practice Questions

  1. Find the derivative of x³.
  2. Differentiate sin(x).
  3. Evaluate ∫3x² dx.
  4. Find lim (x→1) (x²+2).
  5. State the three conditions for continuity.
