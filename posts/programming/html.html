HTML Basics

Learn how websites are built using HTML5.

Learning Objectives

  • Understand what HTML is.
  • Create your first web page.
  • Learn HTML tags and elements.
  • Understand headings and paragraphs.
  • Add links and images.

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It provides the structure of a website using elements called tags.

Your First HTML Page

Explanation

  • <!DOCTYPE html> tells the browser this is an HTML5 document.
  • <html> is the root element.
  • <head> contains information about the page.
  • <body> contains the visible content.
